# Paper 3 notes

Paper 3 focuses on the composition dependence of the fast / NCL-like mechanical-loss response in Cu-Zr metallic glasses.

The first goal is not to change the MD-DMS protocol, but to reuse the Paper 2 protocol across several compositions.

Main MVP comparison:

Cu36Zr64 → Cu50Zr50 → Cu64Zr36

Primary observables:

- storage modulus G'
- loss modulus G''
- tan(delta)
- phase lag
- hysteresis loop shape
- cycle-by-cycle stability
